# Application:
OI Countdown

# Promo text:
Establezca cuentas atrás con OI Countdown.

# Description:


OI Countdown es un temporizador útil para incrementar su productividad y planificar su día. Le permite establecer diversas acciones para notificarle el fin de un temporizador. Personalice completamente sus temporizadores, sin ninguna limitación en cuántos puede tener guardados o en ejecución. 

Esta aplicación no tiene anuncios y no requiere permisos de internet. 





El código fuente de esta aplicación libre y de código abierto está disponible en: 
http://code.google.com/p/openintents

Para la lista completa de cambios y la lista de preguntas frecuentes, por favor visite: 
http://www.openintents.org



Usted puede ayudar a mejorar la traducción de su idioma en Launchpad: 
https://translations.launchpad.net/openintents/trunk

Características: 
* Establezca una cuenta atrás escogiendo la duración o una fecha.
* Sea notificado por vibración o por el sonido que escoja.
* Tenga un número ilimitado de temporizadores ejecutándose al mismo tiempo.
* Guarde sus cuentas regresivas y déles nombre para reutilizarlas más tarde.
* Modifique las cuentas regresivas, incluso cuando estén ejecutándose

Extensiones disponibles para "OI Countdown": 
* Ejecute aplicaciones a través de los accesos directos de Android
* Emplee OI About para mostrar la información de la licencia
