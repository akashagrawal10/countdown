# Application:
OI Χρονομετρητής

# Promo text:
Ορίστε πολλαπλές αντίστροφες μετρήσεις με το OI Χρονομετρητή.

# Description:


Ο OI Χρονομετρητής είναι ένα χρήσιμο εργαλείο για να αυξήσετε την παραγωγικότητα σας και να σχεδιάσετε την ημέρα σας. Σας επιτρέπει να ορίσετε μια ποικιλία διαφορετικών ενεργειών για να σας ενημερώσουν στη λήξη ενός  χρονομετρητή. Προσαρμόστε πλήρως όλους τους χρονομετρητές σας, χωρίς κανένα όριο στους πόσους μπορείτε να έχετε ενεργούς ή αποθηκευμένους. 

Αυτή η εφαρμογή δεν έχει διαφημήσεις και δεν απαιτεί άδεια στο διαδίκτυο. 





Ο πηγαίος κώδικας αυτού είναι δωρεάν και η εφαρμογή ανοικτού κώδικα είναι διαθέσιμη στο: 
http://code.google.com/p/openintents

Για την πλήρη λίστα των αλλαγών και μια λίστα συχνών ερωτημάτων, παρακαλώ επισκευτείτε: 
http://www.openintents.org



Μπορείτε να βοηθήσετε στη βελτίωση της μετάφρασης στη γλώσσα σας στο Launchpad: 
https://translations.launchpad.net/openintents/trunk

Χαρακτηριστικά: 
* Ορίστε μια αντίστροφη μέτρηση επιλέγοντας μια διάρκεια ή μια ημερομηνία.
* Ενημερωθείτε μεσω δόνησης ή κάποιου ήχουν της επιλογής σας.
* Έχετε έναν ατελείωτο αριθμό χρονομετρητών που είναι ενεργοί την ίδια στιγμή.
* Αποθηκεύστε τις αντίστροφες μετρήσεις σας και δώστε τις ονόματα για να τις χρησιμοποιήσετε μελλοντικά.
* Επεξεργαστείτε αντίστροφες μετρήσεις, ακόμα και όταν είναι ενεργές.

Διαθέσιμες επεκτάσεις για το "OI Χρονομετρητής": 
* Εκτελέστε εφαρμογές μέσω τις συντομεύσεις του Android
* Χρησιμοποιήστε το OI Σχετικά για να προβάλλετε πληροφορίες άδειας

Λέξεις κλειδιά: 
* αντίστροφη μέτρηση, χρονομετρητής, ειδοποίηση
* OI, OpenIntents, Open Intents
