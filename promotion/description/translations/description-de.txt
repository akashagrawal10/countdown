# Application:
OI Countdown

# Promo text:
Mit OI Countdown können Sie mehrere Countdowns erstellen.

# Description:


OI Countdown hilft Ihnen, ihren Tag zu planen und Ihre Produktivität zu erhöhen. Sie können festlegen, welche Aktion beim Ende eines Timers geschieht. Sie können unendliche viele Timer erstellen und jeden Timer beliebige Eigenschaften zuweisen. 

Diese Applikation ist werbefrei und benötigt keine Internetverbindung. 





Der Quellcode dieser kostenlosen und quelloffenen Applikation ist verfügbar auf: 
http://code.google.com/p/openintents

Für eine vollständige Liste der Änderungen und häufig gestellten Fragen besuchen sie bitte: 
http://www.openintents.org



Sie können uns mit der Verbesserung der Übersetzung in ihre Sprache auf Launchpad unterstützen: 
https://translations.launchpad.net/openintents/trunk

Leistungsmerkmale: 
* Sie können einen Countdown einrichten, indem Sie die Dauer oder ein Datum festlegen.
* Sie können über das Ende eines Timers mit einen Sound ihrer Wahl oder mit Vibration informiert werden.
* Zur gleichen Zeit können unbegrenzt viele Timer laufen.
* Geben Sie den Countdown beim Speichern einen Namen, damit Sie Ihn später wiederfinden.
* Sie können die Eigenschaften der Countdowns ändern, auch wenn die Countdowns schon laufen

Existierende Erweiterungen für "OI Countdown" 
* Sie können Anwendungen durch Android Shortcuts starten
* Mithilfe von OI Info können Sie Lizentinformationen anzeigen.
