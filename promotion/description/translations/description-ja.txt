# Application:
OI カウントダウン

# Promo text:
OI カウントダウン で複数のカウントダウンを設定できます。

# Description:


OI カウントダウン は効率を上げて一日の計画を立てる役に立つタイマーです。 タイマー終了時に様々なアクションを設定して通知することができます。 タイマーの動作数や保存数の制限無しで、完全にすべてのタイマーをカスタマイズすることができます。 

このアプリケーションは広告を含まず、インターネット許可を必要としません。 





この無料でオープンソースのアプリケーションのソースコードは以下の場所から取得できます： 
http://code.google.com/p/openintents

すべての変更リストや質問リストを見るには： 
http://www.openintents.org



あなたもこの言語の翻訳を Launchpad で改善することができます： 
https://translations.launchpad.net/openintents/trunk

機能： 
* 期間か日付を選択することでカウントダウンをセットします。
* あなたの選択で振動かサウンドで通知されることができます。
* 同時に走る
* カウントダウンを保存して後で再開できるように名前を付けます。
* カウントダウンの変更（カウントダウンも含む）

"OI カウントダウン" の利用可能な拡張子： 
* アンドロイドショートカットからアプリケーションを起動する
* OI 情報 を使ってライセンス情報を表示

キーワード: 
* カウントダウン、 タイマー、 警告
* OI, OpenIntents, Open Intents
